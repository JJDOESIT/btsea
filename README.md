# BTSea
<b>Online Functional Bitcoin Wallet:<b>
<br>
Built using React + Django + Django Rest Framework + Bitcoinlib
<br>
<b>Link:<b> https://jamesgwhit.pythonanywhere.com/

# Features
- User Login + Registration
- 2FA Email Authentication
- Send and receive Bitcoin on the Testnet network
- Analytics chart on the price of Bitcoin
- Data cards on the trending cryptocurrencies
